With the rapid growth of cloud computing and the transition of applications towards microservices architectures, there is an increasing demand for efficient and scalable resource management strategies. This paper introduces a novel approach that utilizes machine learning algorithms to dynamically scale microservices in response to varying workloads. Introduction

The landscape of software engineering has witnessed a significant evolution with the advent of microservices architecture, which gradually replaces monolithic systems across diverse sectors. Microservices offer a decoupled approach that allows distinct components of applications to be developed, deployed, and scaled independently, enhancing flexibility and fault tolerance. However, this rise in microservices usage, especially in cloud environments, poses new challenges particularly in resource management and scalability. Traditional auto-scaling methods often rely on static threshold-based policies which can lead to suboptimal resource utilization, potentially increasing costs and deteriorating user experience during peak incident demand. Static methods lack the ability to respond proactively to unpredictable workload changes typical in modern applications. Addressing these drawbacks calls for more sophisticated mechanisms that can adaptively scale resources in real-time. Machine learning provides promising solutions through predictive analytics and dynamic model training. By forecasting resource usage based on historical and real-time data, machine learning models can automate the scaling process with higher accuracy and efficiency. Our study focuses on embedding machine learning techniques within microservices environments to form an adaptive scaling framework. This framework continuously learns and predicts demands, ensuring resources align precisely with requirements, minimizing both latency and operating costs. In this paper, we explore previous literature highlighting existing methodologies used for dynamic scaling in cloud platforms. We identify limitations in current approaches and propose an intelligent scaling system engineered to overcome these challenges. Our system is extensively tested across varied workloads and cloud configurations, demonstrating significant improvements over traditional approaches. This contribution is aimed at developers and architects seeking to enhance the performance and economics of their cloud-based systems by embracing adaptive, intelligent strategies for resource management in microservices architectures.
